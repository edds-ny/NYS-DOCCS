## âš™ï¸ Installation & Deployment

This project is built as a standalone HTML file, making it perfect for static hosting like GitHub Pages.

### Option 1: GitHub Pages (Recommended)

1.  Create a new repository on GitHub.
2.  Upload the `index.html` file to the root of your repository.
3.  Go to **Settings** > **Pages**.
4.  Under **Build and deployment** > **Branch**, select `main` (or `master`) and `/` (root).
5.  Click **Save**. Your site will be live in minutes.

### Option 2: Local Development

To run this project locally, clone the repository and open the file in your browser.
